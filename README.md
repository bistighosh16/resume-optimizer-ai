# рҹҡҖ Resume Optimizer AI

An AI-powered web application that analyzes your resume against any job description, generates a complete optimized resume tailored to the role, and lets you download it as a polished PDF вҖ” all powered by **LLaMA 3.1** via Groq's lightning-fast inference API.

> Built with вқӨпёҸ to help job seekers land their dream roles.

## вңЁ Features

### рҹҺҜ Smart Analysis
- рҹ“Ҡ **Animated Match Score** вҖ” Visual circular progress bar showing your resume-job alignment (0вҖ“100%)
- вң… **Matched Keywords** вҖ” Color-coded pills showing skills/keywords already in your resume
- вқҢ **Missing Keywords** вҖ” Identify critical keywords from the job description you're missing
- рҹ’Ә **Strengths Highlight** вҖ” AI identifies your top selling points for the role
- рҹҺҜ **Skills Gap Analysis** вҖ” Detailed breakdown of what you need to improve

- вңЁ **Complete Resume Rewrite** вҖ” One-click generation of a fully optimized resume tailored to the target job
- рҹҺЁ **Beautiful Preview** вҖ” Clean, professional formatting with sections for Summary, Skills, Experience, Education, and Projects
- рҹ“Ҙ **PDF Download** вҖ” Export your optimized resume as a high-quality PDF ready to send to recruiters

### рҹҺЁ User Experience
- рҹҢҷ **Dark/Light Mode** вҖ” Toggle between themes with preference saved in browser
- рҹ“ұ **Fully Responsive** вҖ” Works beautifully on desktop, tablet, and mobile
